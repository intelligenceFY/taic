# TaiBreadcrumb

这是 TaiBreadcrumb 的 demo

```jsx
import { TaiBreadcrumb } from 'taic';
const item =  [
      {
        title: 'Home',
      },
      {
        title: 'Application Center',
        href: '',
      },
      {
        title: 'Application List',
        href: '',
      },
      {
        title: 'An Application',
      },
    ]

export default () => <TaiBreadcrumb  items={item}/>;
```
